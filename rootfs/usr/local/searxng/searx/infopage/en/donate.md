# Make a donation

This instance of SearXNG is powered by the [nedix/searxng (Docker Hub)](https://hub.docker.com/r/nedix/searxng) container. Unlike the standard [searxng/searxng (Docker Hub)](https://hub.docker.com/r/searxng/searxng) container the following additional features are included:

- bot detection and mitigation
- configuration through environment variables
- whitelist approach to engine and plugin configuration
